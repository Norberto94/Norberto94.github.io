1. Function Calls 1. Parentheses can sometimes be omitted. If you're not sure whether they're required, put them in. To be safe, put them in whenever the call is at all complicated. Even one as simple as this. 1. puts "hello" puts("hello")
2. Variables 2. Ordinary (local) variables are created through assignment: 2. number = 5
3. Strings 3. Strings are sequences of text in quotation marks. You can use single or double quotes. 3. name = 'simon'
name = "simon"
4. Methods 4. A method call looks like this: 4. "simon".upcase # returns "SIMON"
5. Conditionals (if) 5. Put the if, elsif, else, and end on separate lines as shown. You don't have to indent, but you should. 5. if number == 5
puts "Success"
elsif number == 6
puts "Acceptable"
else
puts "FAILURE"
end
6. Function Definitions 6. Functions can return values, and those values can be assigned to variables. The return value is the last statement in the definition. Here's a simple example: 6. def assert_equal(expected, actual)
if expected != actual
puts "FAILURE!"
end
end
7. Here's a little more complicated example: 7. def make_positive(number)
if number < 0
-number
else
number
end
end
 7. variable = make_positive(-5) # variable's value is 5
variable = make_positive(five) # variable's value is 5
8. Arrays 8. This is an array with nothing in it: 8. []
9. More arrays 9. This is an array with two numbers and a string in it. You can put anything into an array. 9. [1, 'hello!', 220]
10. More arrays 10. Here's how you get something out of an array: 10. array = [1, 'hello', 220]
array[0] # value is 1
11. More arrays 11. How long is an array? 11. array.length
13. More arrays 13. Here's how you change an element: 13. array[0]= 'boo!' # value printed is 'boo!'
# array is now ['boo', 'hello', 220]
14. More arrays 14. Here's how you tack something onto the end of an array: 14. array.push('fred') # array is now ['boo', 'hello', 220, 'fred']
15. More arrays 15. Here's how you add an element at a specific index: 15. array.insert(1, "bye")
# array is now ['boo', "bye" , 'hello', 220, 'fred']
.insert takes two arguments, first the index where you want to insert to and second the thing to be inserted
16. Iteration 16. When you do something multiple times, it is called iteration. There are many ways to do this. The following will print hello five times: 16. 5.times do
puts 'hello'
end
17. More iteration 17. Here's one way to print the numbers from one to 10: 17. for x in 1..10 do
puts x
end
18. More iteration 18. And here's another: (1..10).each do |x|
puts x
end
19. More iteration 19. The part between the do and the end is called a block. You can replace the do and end with braces: 19. (1..10).each { |x| puts x } end
20. More iteration 20. This prints each value of an array: 20. ["a", "b", "c"].each { |x| puts x } end
21. More iteration 21. What if you want to transform each element of an array? The following capitalizes each element of an array. 21. ["hi", "there"].collect { |word| word.capitalize }
# The result is ["Hi", "There"]. end
